# My simple set of PPM utils for writing and reading P6 images.
## How it's made
- No dependencies, except for SDL2 for viewer
- Written in pure C (C11), compiled with GCC using Makefiles generated by CMake
## How to build
1. Install CMake (minimum version 3.10 required)
2. Proceed with general CMake building process
```shell
cd /path/to/project/
mkdir build
cmake ..
cmake --build .
```
## What's included?
- A simple PPM P6 writer
- A simple PPM P6 viewer (made with SDL2)
## What's planned?
- Full support for P3 (non-binary) format of images
- Full support of images with maximal value (maxval) not equal to 255
